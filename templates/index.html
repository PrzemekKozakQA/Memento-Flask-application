"Memento" is a web application that helps you remember definitions or words from foreign languages.
It won't learn for you, but it can help you learn. In "Memento" you can add your words, view a list of all added words and test your memory in a quiz.

All you need is access to the application via a web browser on your computer or mobile device.
Apart from that, there are no other requirements. Just register and start using the application!

Yes, registration is necessary, but it is very quick.
Registration is necessary because it allows each user to have individual words and definitions independent of others.
